查看提交历史[返回]

$git log [options]

没有选项默认打印所有的提交历史

常用选项

$git log -p -2

-p选项会打印每次提交的差异

-n，n为数字，最多打印最近n条提交历史

$git log --pretty=oneline

每个提交历史只显示一行

还有short, full, fuller等可以用

$git log --pretty=format:"%h %s"

格式化提交历史输出，这个指令可以方便变成提取分析

$git log --since="<time-string>" --until="<time-string>"

--before="<time-string>" --after="<time-string>"

可以给出各种时间格式，例如："2013-01-07"，"2 years 1 day 3 minutes ago"

$git log --grep="<grep-string>"

grep提交说明字符串

$git log -- <file_or_dir>

只显示针对指定文件与目录下文件的更新提交历史
